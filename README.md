# CommentX Agent

CommentX Agent 是一个智能浏览器扩展（Chrome Extension），能够在用户浏览微博等社交平台时，自动模拟人工操作进行帖子翻页、点开评论框、填入预设评论草稿，并支持自动或人工审核后发送。

## 🌟 主要功能

- **自动化操作**：自动检测信息流中的新帖子，自动点击评论按钮展开输入框。
- **智能填词**：利用底层 `Native Input Setter` 绕过前端框架限制，完美模拟人工输入，填入随机的预设评论。
- **安全审核模式**：默认开启“需人工确认后发送”，填好草稿后在输入框旁给出醒目提示，由用户亲自点击“发送”，确保内容安全可控。
- **自动翻页**：支持开启自动向下滚动，源源不断地加载新帖子进行处理。
- **任务队列机制**：帖子处理引入队列和延时机制，避免同时展开多个评论框导致页面混乱，行为更像真实用户。

## 📦 安装与使用方法

本项目基于 React + Vite 开发，请按照以下步骤构建并加载到 Chrome 浏览器中：

### 1. 安装依赖并构建
确保你已经安装了 [Node.js](https://nodejs.org/)。

```bash
# 安装依赖
npm install

# 构建扩展 (将会在项目根目录生成 dist 文件夹)
npm run build
```

### 2. 加载到 Chrome 浏览器
1. 打开 Chrome 浏览器，在地址栏输入 `chrome://extensions/` 并回车。
2. 在右上角开启 **“开发者模式”** (Developer mode)。
3. 点击左上角的 **“加载已解压的扩展程序”** (Load unpacked)。
4. 在弹出的文件选择窗口中，选中本项目根目录下的 `dist` 文件夹，点击确定。
5. 安装成功后，浏览器的扩展栏会出现 `CommentX Agent` 的图标（可能需要点击拼图图标将其固定）。

### 3. 开始实测 (以微博为例)
1. 点击浏览器右上角的 `CommentX Agent` 扩展图标，打开控制面板。
2. 在控制面板中配置你的 **“评论模板库”**（可以添加多条，Agent 会随机抽取）。
3. 打开 **“主开关：自动处理”**。
4. （可选）如果你希望它完全全自动，可以关闭“需人工确认后发送”；如果希望自动翻页，可以勾选“自动翻页”。
5. 新开一个标签页，访问 [微博网页版 (weibo.com)](https://weibo.com)。
6. 向下滚动页面，你会看到 Agent 开始自动帮你点击评论框并填入内容！

## 🛠️ 技术栈
- React 19
- Vite
- Chrome Extension Manifest V3
- TypeScript

## 📝 注意事项
- 由于各大社交平台的 DOM 结构可能会随时更新，如果发现无法自动评论，请检查目标平台的页面结构是否发生了变化。
- 建议在测试时保持“需人工确认后发送”处于开启状态，避免因全自动发送触发平台的反垃圾/风控机制。
